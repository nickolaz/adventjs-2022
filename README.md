# 🎅🎄 My AdventJS 2022 solutions 🎄🎅

Solutions to [adventJS-2022](https://adventjs.dev/) by [@midudev](https://twitter.com/midudev).

## 💻️ Installation & usage

- Clone this project
- Install dependencies
- Run all tests or run only the day X tests

```bash
npm install
# Run all tests
npm run test
```
